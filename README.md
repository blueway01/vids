# VEO HUB - Google Veo 3.1 Prompt Library

Google Veo 3.1 および Google Vids 用のプロンプト共有ハブです。

## 特徴
- **カテゴリ別プロンプト**: Cinematic, Corporate, Animation, Nature など目的に合わせたプロンプトを掲載。
- **Google Vids 最適化**: Vidsの「Generate Video」機能や「Ingredients」機能で使いやすいプロンプト構造。
- **コピー＆ペースト**: 英語のプロンプトをワンクリックでコピー可能。

## 技術スタック
- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Icons**: Lucide React

## 開発サーバーの起動
```bash
npm run dev
```

## プロンプトの追加方法
`src/data/prompts.ts` に新しい項目を追加してください。
